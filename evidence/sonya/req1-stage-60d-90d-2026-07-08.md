## SQL — 60-Day Window

```sql
SELECT
  record_id AS campaign_id,
  ROUND(SUM(spend)::numeric, 2)  AS cost,
  ROUND(SUM(sales)::numeric, 2)  AS cv,
  ROUND(SUM(orders)::numeric, 2) AS conv,
  ROUND((SUM(sales)/NULLIF(SUM(spend),0)*100)::numeric, 2) AS roas
FROM public.ppc_etl_performance_data
WHERE record_id IN (
  '20810136438','21435967873','22943583032','23526695953',
  '23914872425','23515806682','23729304135','22847654610','23793722836'
)
AND record_type = 'campaign' AND source = 3
AND date >= '2026-05-09' AND date <= '2026-07-07'
GROUP BY record_id
ORDER BY cost DESC;
```
