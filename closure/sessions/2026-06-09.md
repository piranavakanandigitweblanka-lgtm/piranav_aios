| P1-B | `snippets/minicart.liquid` line 1 | Removed duplicate cart.css load | Eliminates double parse |
| P1-C | `snippets/pendantoffpop.liquid` line 36 | Removed `@import url(google fonts)` — worst loading pattern | Eliminates CSS parse block |

| P2-C | `sections/main-product.liquid` line 3 | glightbox.min.css now async on product pages | 16 KB off PDP blocking path |

#### Evidence
All 6 fixes verified via grep check — correct patterns confirmed in each file.
