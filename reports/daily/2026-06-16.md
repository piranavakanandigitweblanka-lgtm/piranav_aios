### blueskytechco.woff — Unused Font Preload Fix
- **File modified:** `layout/theme.liquid` line 222
- **Root cause:** Preload used bare URL `blueskytechco.woff`; `@font-face` in `vendor.css` loads versioned URL `blueskytechco.woff?s9sn20` — browser treats these as different URLs, preload is never consumed
- **Secondary issue:** `@font-face` has no `.woff2` format; modern browsers may not use `.woff` at all
- **Fix:** Removed preload tag — font continues loading correctly via `vendor.css` `@font-face`
- **Font still active:** Yes — used in 9 CSS files as icon font via `--font-icon` CSS variable
- **Risk:** GREEN — one `<link>` tag replaced with comment, no visual change
