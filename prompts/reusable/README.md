# prompts/reusable/ — Cross-Domain Reusable Prompts

Prompt templates that apply across multiple task categories (discovery + implementation, or implementation + validation) and cannot be cleanly placed in a single category folder.

## Examples
- Patterns used for both auditing AND fixing (e.g. breadcrumb audit that includes the fix steps)
- Prompts that span multiple sessions or stores
- Composite patterns combining 2+ task types
